# إعداد الشعار - Logo Setup

## لإضافة الشعار للموقع:

### الخطوة 1: حفظ ملف الشعار
1. احفظ صورة الشعار باسم `logo.png` 
2. ضعها في المجلد: `public/images/logo.png`

### الخطوة 2: متطلبات الصورة
- **الصيغة**: PNG أو JPG
- **الحجم المُوصى به**: 400x200 بكسل
- **خلفية شفافة**: مُفضل للـ PNG
- **جودة عالية**: للحصول على أفضل مظهر

### الخطوة 3: التحقق من التشغيل
بعد وضع الصورة في المكان الصحيح، سيظهر الشعار في:
- ✅ **الهيدر الرئيسي** للموقع
- ✅ **صفحة تسجيل الدخول**
- ✅ **في حالة فشل تحميل الصورة** سيظهر النص كبديل

### الخطوة 4: اختبار الشعار
```bash
npm start
```
وتحقق من ظهور الشعار في الموقع.

## Logo Setup Instructions

### Step 1: Save Logo File
1. Save the logo image as `logo.png`
2. Place it in the folder: `public/images/logo.png`

### Step 2: Image Requirements
- **Format**: PNG or JPG
- **Recommended Size**: 400x200 pixels
- **Transparent Background**: Preferred for PNG
- **High Quality**: For best appearance

### Step 3: Verify Installation
After placing the image in the correct location, the logo will appear in:
- ✅ **Main header** of the website
- ✅ **Login page**
- ✅ **Fallback text** if image fails to load

### Step 4: Test Logo
```bash
npm start
```
and verify the logo appears on the website.
